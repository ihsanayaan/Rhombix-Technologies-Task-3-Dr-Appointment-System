Booking System UI (Internship Task  Demo Project)

This repository contains a demo Booking System UI created as part of an internship task.
The project focuses on UI/UX design thinking and frontend implementation, rather than backend or real-world data.

The complete flow was first designed as wireframes in Figma and then converted into a fully responsive React application.

Project Objective

Design a complete booking system user flow 

Apply UI/UX principles in a real interface

Convert Figma wireframes into React components

Build a mobile first responsive demo application

Note: This is a UI demo project.
No real backend, authentication, or payment system is implemented.

Features

Home screen with featured services & categories

Services listing with pricing, category & duration

Appointment booking flow (demo)

Booking summary screen

Profile UI & navigation drawer

Mobile bottom navigation + desktop sidebar

Tech Stack

React.js (Vite)

Tailwind CSS

React Router DOM

Zustand (state management)

React Hook Form

Lucide React Icons

Figma (Wireframes & UI flow)
