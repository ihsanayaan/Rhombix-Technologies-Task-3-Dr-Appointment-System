#  Healthcare Booking System

Modern bilingual healthcare booking platform with complete RTL/LTR support built using React.js, Tailwind CSS, Zustand, and i18next.

The complete flow was first designed as wireframes in Figma and then converted into a fully responsive React application.

Project Objective

Design a complete booking system user flow 

Apply UI/UX principles in a real interface

Convert Figma wireframes into React components

Build a mobile first responsive demo application
